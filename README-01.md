# 依赖

- 编译框架：`meson`
  - 可通过 `gcc -lm main.c hw01.c -Wall`编译
- 编译器：在`GCC 14.2.1`, `CLANG 18.1.8`下测试通过

# 使用

1. 程序将等待用户输入操作。
2. 可用的操作包括：
   - `F <x>`：计算并打印函数 `calc_func` 在 `x` 处的值。
   - `Y <年份>`：检查指定的年份是否为闰年。
   - `P <p>`：计算并打印基于初始金额 `p` 的不同利润率。
   - `d <p>`：设置活期利润率为 `p`。
   - `f <p>`：设置固定利润率为 `p`。
   - `t <p>`：设置固定两次利润率为 `p`。
   - `q`：退出程序。
   - `h` 或 `?`：显示帮助信息。
     例如，要检查 2020 年是否为闰年，输入：

```
> Y 2020
2020 is a leap year
```
